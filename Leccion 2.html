Leccion 2: Tema 2 - Variables y tipos de datos

Variables y Tipos de Datos

En programación, las variables son contenedores para almacenar datos. Los tipos de datos más comunes son:

  • Números: enteros y decimales. Ejemplo: int, float
  • Cadenas de texto: secuencias de caracteres. Ejemplo: str
  • Booleanos: valores de verdadero o falso. Ejemplo: bool
  • Listas: colecciones ordenadas y modificables de elementos. Ejemplo: list
  • Tuplas: colecciones ordenadas e inmutables de elementos. Ejemplo: tuple
  • Conjuntos: colecciones desordenadas de elementos únicos. Ejemplo: set
  • Diccionarios: colecciones de pares clave-valor. Ejemplo: dict

Operadores Aritméticos y de Asignación

Los operadores aritméticos se utilizan para realizar operaciones matemáticas, como suma, resta, multiplicación, división, etc. Los operadores de asignación se utilizan para asignar valores a variables.

Ejemplos de operadores aritméticos:

  • Suma: +
  • Resta: -
  • Multiplicación: *
  • División: /
  • Módulo: %
  • Potencia: **

Ejemplos de operadores de asignación:

  • Asignación simple: =
  • Suma y asignación: +=
  • Resta y asignación: -=
  • Multiplicación y asignación: *=
  • División y asignación: /=
  • Módulo y asignación: %=
  • Potencia y asignación: **=

Estructuras de Control

Las estructuras de control permiten controlar el flujo de ejecución de un programa. Algunas estructuras de control comunes son:

  • Condicionales if-else: permiten ejecutar diferentes bloques de código dependiendo de una condición.
  • Bucles for: permiten iterar sobre una secuencia de elementos un número determinado de veces.
  • Bucles while: permiten ejecutar un bloque de código mientras se cumpla una condición.

Funciones y Módulos

Las funciones son bloques de código reutilizables que realizan una tarea específica. Los módulos son archivos que contienen funciones y otros elementos relacionados que pueden ser importados y utilizados en otros programas.

Para definir una función en Python:

def nombre_funcion(parametros):
  # Código de la función
  return resultado

Ejemplo de uso de una función:

def suma(a, b):
  return a + b

resultado = suma(3, 5)
print(resultado)  # Output: 8

Para importar un módulo en Python:

import nombre_modulo

resultado = nombre_modulo.nombre_funcion(parametros)
