Lección 4: Lectura y Escritura de Archivos, CSV y JSON, y Manejo de Errores

Lectura y Escritura de Archivos de Texto

Python proporciona funciones para leer y escribir archivos de texto.

# Abrir un archivo en modo escritura
archivo = open("archivo.txt", "w")

# Escribir en el archivo
archivo.write("Hola, mundo!")

# Cerrar el archivo
archivo.close()

# Abrir un archivo en modo lectura
archivo = open("archivo.txt", "r")

# Leer el contenido del archivo
contenido = archivo.read()
print(contenido)  # "Hola, mundo!"

# Cerrar el archivo
archivo.close()

Manipulación de Archivos CSV y JSON

Python ofrece módulos integrados para trabajar con archivos CSV y JSON.

Archivos CSV

El módulo `csv` proporciona funciones para leer y escribir archivos CSV.

import csv

# Leer un archivo CSV
with open("datos.csv", "r") as archivo_csv:
    lector = csv.reader(archivo_csv)
    for fila in lector:
        print(fila)

# Escribir en un archivo CSV
with open("datos.csv", "w") as archivo_csv:
    escritor = csv.writer(archivo_csv)
    escritor.writerow(["Nombre", "Edad"])
    escritor.writerow(["Juan", 25])
    escritor.writerow(["María", 30])

Archivos JSON

El módulo `json` proporciona funciones para leer y escribir archivos JSON.

import json

# Leer un archivo JSON
with open("datos.json", "r") as archivo_json:
    datos = json.load(archivo_json)
    print(datos)

# Escribir en un archivo JSON
datos = {"nombre": "Juan", "edad": 25}
with open("datos.json", "w") as archivo_json:
    json.dump(datos, archivo_json)

Excepciones y Manejo de Errores

En Python, podemos manejar errores y excepciones utilizando bloques `try-except`.

try:
    # Código que puede generar un error
    resultado = 10 / 0
except ZeroDivisionError:
    # Manejo del error
    print("Error: división por cero")

En el ejemplo anterior, intentamos dividir 10 entre cero, lo cual generará un error de división por cero. Utilizamos el bloque `try-except` para capturar el error y mostrar un mensaje personalizado.
